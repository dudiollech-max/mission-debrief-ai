# Mission Debrief AI
### Edge AI Recorder + Auto-Debrief for Unmanned Systems

> **VisionWave AI Asset — Product 5**  
> Prepared for VisionWave Holdings, Inc. (Nasdaq: VWAV)

---

## What It Does

An edge-deployable AI agent that ingests multi-stream mission data — video, telemetry, sensor logs — and produces a **structured natural-language debrief in under 60 seconds**.

- ✅ Flags anomalies automatically
- ✅ Summarises mission timeline
- ✅ Captures key decision points
- ✅ Output: PDF or JSON for downstream systems

---

## Use Cases

- **Argus** — Counter-UAS debrief
- **UGV** — Ground vehicle mission review
- **SolarDrone** — Solar O&M inspection debrief (real customer data)

---

## AI Core

- Open-source multi-modal LLM (Qwen2-VL / Llama 3.2 Vision class)
- Anomaly-detection layer on telemetry streams
- Auto-flagged "interesting moments" with thumbnail evidence
